Change the style of your posts with options drop down menu.
You can also implement a blog-azine with a different style in every post.

== Description ==

This plugin helps you simplify replacement (the template single post) with a dropdown menu.

You must create a new template with a structure and a single post with a different style.
If no Single post template is selected, the default template will be used.

== Installation ==

**Installation**

1. Upload the entire `single-post-template` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create Post Templates in your theme's folder.

**Usage**

After you have installed the plugin, you'll need to create some Post Templates with a structure and a single post with a different style to use. Insert the following code at the very top of the file:

`
<?php
/*
Style : [Your Template Name]
*/
?>
`
